Chicago Style Notes

NB System

예술·인문학 분야 표준. 각주/미주 + 참고문헌 목록의 3요소로 구성됩니다. 사회과학에서는 Author-Date 방식을 더 자주 사용합니다.

17판 변경점 · Ibid.

"Ibid."의 사용은 지양됩니다. 반복 인용 시 저자 성 + 축약 제목(4단어 이하) + 페이지 형식을 사용하세요.

17판 변경점 · 웹사이트명

웹사이트명은 인쇄판이 존재하는 경우에만 이탤릭체로 표기합니다. 그 외 모든 경우 기본 로마체로 유지하세요.

Accessed Date

출판일/개정일이 없는 웹 자료에 한해 접속일(accessed date)을 표기합니다. 출판일이 있으면 생략하는 것이 원칙입니다.
